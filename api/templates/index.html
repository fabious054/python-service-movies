Welcome to JUSTMOVIEIT API service

This Python API service, created by Fabio Abreu, will be instrumental in building my own movie community. By querying the TMDB database and importing the data into my own repository, I'll establish a platform where I can exert precise control over the movies, adapting them to the specific needs and ideas of my community.

Routes

  • /movies

    This route will retrieve all movies from TMDB within the range from 2000 to 2030. The search will provide the IDs for each year divided into monthly queries to avoid overloading. These IDs will be checked against my database, and if they don't exist, I will add them.

  • /genres

    This route will fetch the list of all existing genres in the database, which will be linked to the movies. The query verifies if the genre exists in my database, and if it doesn't, the genre will be added.

  • /providers

    This route consults all providers registered in the TMdb database and adds them to my database if they do not exist.

Config

  • For the routes to function properly, adjustments to the project's environment variables are necessary. So, running the project without making these adjustments beforehand won't be effective.
  • In the future, I plan to make it available for anyone interested to export the data to their own databases.
